# Redactly

A simple browser-based PDF redaction tool. Upload a PDF, cover sensitive areas, remove pages if needed, and export a fresh copy.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Then open `http://localhost:3000`.

## Scripts

- `pnpm dev` starts the local Next.js app.
- `pnpm build` builds the app for production.
- `pnpm lint` runs ESLint.
